Ergani Schedule Manager Documentation

Comprehensive guide to installing, configuring, and using the Ergani Schedule Manager application.

Introduction

Ergani Schedule Manager is a desktop application designed to simplify the process of creating, managing, and submitting employee schedules to the Ergani system. Built with modern web technologies and packaged as a desktop application, it provides a user-friendly interface for managing all aspects of employee scheduling.

This application is designed to work with the Ergani system used in Greece for reporting employee schedules and work hours.

Installation

The application is available for Windows, macOS, and Linux. Download the appropriate installer for your operating system from the download page.

Windows

  1. Download the Windows installer (.exe file)
  2. Run the installer and follow the on-screen instructions
  3. The application will be installed and a shortcut will be created on your desktop

macOS

  1. Download the macOS disk image (.dmg file)
  2. Open the DMG file and drag the application to your Applications folder
  3. Right-click on the application and select "Open" the first time you run it (this bypasses macOS security restrictions)

Linux

  1. Download the AppImage file
  2. Make it executable: chmod +x Ergani-Schedule-Manager-*.AppImage
  3. Run the application: ./Ergani-Schedule-Manager-*.AppImage

Getting Started

When you first launch the application, you will need to configure your Ergani credentials and set up your organization profile.

Initial Setup

  1. Launch the application
  2. Go to the Settings page by clicking the gear icon in the sidebar
  3. Enter your Ergani credentials in the form provided
  4. Save your settings

Creating Your First Schedule

  1. Click the "New Schedule" button on the home screen
  2. Select the date range for the schedule
  3. Add employees to the schedule by clicking the "Add Employee" button
  4. Assign shifts to employees by clicking on the corresponding cells in the schedule grid
  5. Save the schedule by clicking the "Save" button

Features

  • Employee Management: Add, edit, and manage employee records including contact information, schedules, and work history.
  • Schedule Creation: Create weekly or monthly schedules with a drag-and-drop interface.
  • Ergani Integration: Submit schedule data directly to the Ergani system with a single click.
  • Reporting: Generate reports on employee work hours, absences, and more.
  • Data Export: Export schedules and reports to various formats including PDF, Excel, and CSV.
  • Automatic Updates: The application will automatically check for and install updates to ensure you always have the latest features and bug fixes.

Configuration

The application stores configuration data in the following locations:

  • Windows: %APPDATA%\ergani-schedule-manager\config.json
  • macOS: ~/Library/Application Support/ergani-schedule-manager/config.json
  • Linux: ~/.config/ergani-schedule-manager/config.json

You can edit this file directly, but it's recommended to use the Settings interface in the application.

Troubleshooting

If the application won't start, try the following:

  1. Make sure you have the latest version of the application
  2. Try reinstalling the application
  3. Check your system's requirements
  4. Look for error logs in the following locations:
    • Windows: %APPDATA%\ergani-schedule-manager\logs
    • macOS: ~/Library/Logs/ergani-schedule-manager
    • Linux: ~/.local/share/ergani-schedule-manager/logs

If you can't connect to the Ergani system, check the following:

  1. Verify your internet connection
  2. Check that your Ergani credentials are correct in the Settings page
  3. Make sure the Ergani system is not undergoing maintenance
  4. Check your firewall settings to ensure the application has internet access

API Reference

The application includes a Python backend with the following API endpoints:

GET  /api/employees         - Get all employees
POST /api/employees         - Create a new employee
GET  /api/employees/:id     - Get a specific employee
PUT  /api/employees/:id     - Update an employee
DEL  /api/employees/:id     - Delete an employee

GET  /api/schedules         - Get all schedules
POST /api/schedules         - Create a new schedule
GET  /api/schedules/:id     - Get a specific schedule
PUT  /api/schedules/:id     - Update a schedule
DEL  /api/schedules/:id     - Delete a schedule

POST /api/ergani/submit     - Submit data to Ergani
GET  /api/ergani/status     - Check Ergani connection status

These endpoints are used internally by the application and are not intended for external use.
